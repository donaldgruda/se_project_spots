# Project 3: Spots

## Description

Spots is a responsive photo-sharing web application that displays a user profile and a grid of photo cards. The project was built from a Figma design specification, focusing on creating a pixel-perfect implementation that adapts seamlessly across desktop, tablet, and mobile devices. Users can view a profile with an avatar, name, and description, along with a gallery of location photos.

## Tech Stack

- **HTML5** - Semantic markup and structure
- **CSS3** - Styling and layout
- **Flexbox** - Profile section layout and button alignment
- **CSS Grid** - Responsive photo card gallery with auto-fit
- **Media Queries** - Responsive design for mobile and intermediate screen sizes
- **BEM Methodology** - CSS class naming convention for maintainable code
- **Figma** - Design specifications and assets
- **Git & GitHub** - Version control and deployment

## Features

- Fully responsive design (320px to 1440px+)
- Text overflow handling with ellipsis for long titles
- Hover states for interactive elements
- Centered layout with maximum content width of 1280px
- Mobile-first approach with progressive enhancement
